<p>
In dit project werken we aan een SDG game, in opdracht van NVVN. SDG game, (wat oorspronkelijk een bord spel is) is een spel om mensen bewust te maken van het milieu. 

Onze taak hierbij is om met SVG elementen die we mee hebben gekregen een slide-transitie te maken, die vervolgens gebruikt kan worden in het spel wanneer je bijv. op een level drukt. Dit willen we doen door middel van LottieFiles.
